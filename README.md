# 📌 Task Manager Backend (Node.js + MongoDB)

A simple REST API for managing tasks using Node.js, Express, and MongoDB.  
This project is built for learning CRUD operations in backend development.

---

## 🚀 Main Features

- Create, Read, Update, Delete (CRUD) tasks
- Task status: pending / completed / cancelled
- MongoDB database integration
- Input validation with Mongoose
- Timestamps (createdAt, updatedAt)

---

## 📱 Development Info (IMPORTANT)

This project is fully developed on an **Android device** using:

- 📱 Termux (Node.js environment)
- ✍️ Acode Editor (code editor)

👉 No laptop or PC used — only mobile development.
